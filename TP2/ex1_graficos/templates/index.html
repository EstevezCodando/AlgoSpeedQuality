Quantidade de Passos por Entrada

Função 1 - O(n)

Esta função realiza uma operação de loop que itera de 0 até n - 1.

Para cada iteração, uma operação de complexidade O(1) é realizada.

Portanto, a complexidade total é O(n), pois o loop é executado n vezes.

def funcao1(n):
    for i in range(n):
        print("Iteração:", i) # O(1) por iteração

Função 2 - O(n²)

Esta função possui dois loops aninhados. O loop externo itera de 0 até n - 1, e o loop interno também itera n vezes.

A cada iteração, uma operação de complexidade O(1) é realizada.

Como o loop interno é executado n vezes para cada iteração do loop externo, a complexidade é O(n²).

def funcao2(n):
    for i in range(n):
        for j in range(n):
            print("Iteração:", i, j) # O(1) por iteração do loop interno

Função 3 - O(2n)

Esta função calcula a sequência de Fibonacci de forma recursiva.

Cada chamada faz duas novas chamadas recursivas, resultando em uma árvore de chamadas de complexidade exponencial.

A complexidade final é O(2n).

def funcao3(n):
    if n <= 1:
        return n
    else:
        return funcao3(n - 1) + funcao3(n - 2)
